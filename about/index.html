    Add preview of pdf

    See this related post on stackoverflow.

    Instructions:

    • Select the pdf file you want to embed
    • Right click > Preview > More actions > Open in a new window. Now click More actions > Embed item
    • Copy the embed code and paste it and change the last part from view to preview (File must be set publicly visible if you want everyone to see this)
    <iframe src="https://drive.google.com/file/d/0B3VnSl89GUNxaXI1NzNrLUZTYWs/preview" width="640" height="480"></iframe>

    Add frame of GoogleDoc

    It is possible to modify the google docs files from a frame in the website by adding a frame element following these instructions.

    Instructions:

    • Open the document file you want to embed
    • Copy the link of the document
    • Change the last part of the link

    Note

    For example this link:

    https://docs.google.com/document/d/1bQrWnO1x3lV_tLf4VYdkHOZqw0v8g-jcLJxGY-TPsOM/edit

    become:

    https://docs.google.com/document/d/1bQrWnO1x3lV_tLf4VYdkHOZqw0v8g-jcLJxGY-TPsOM/view?format=pdf&embedded=true

    • Copy the code as a frame object
    <iframe src="https://docs.google.com/document/d/1bQrWnO1x3lV_tLf4VYdkHOZqw0v8g-jcLJxGY-TPsOM/view?format=pdf&embedded=true" 
style="width:900px; height:1000px;" frameborder="0"></iframe>
